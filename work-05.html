05 — UI / UX Design

Mono

「書く」という行為に集中するためのジャーナリングアプリ。余計な機能を削ぎ落とし、思考と向き合うための静かな空間をデザインした。

Client

Mono App

Year

2022

Role

UI / UX Design

Tools

Figma, Protopie, Swift UI

Overview

現代のノートアプリは機能が多すぎる。Monoは「引き算のデザイン」で、書くことだけに集中できるジャーナリングアプリ。タグもフォルダもない。あるのはテキストと日付だけ。

ユーザーリサーチで発見した「書き始めの壁」を解消するUIを設計した。

Approach

モノクロームのインターフェースで視覚的なノイズを徹底排除。起動から即入力のワンステップ設計。タイポグラフィに拘り、日本語・英語の混植でも美しく見えるフォント設定を追求した。

カレンダービューは書いた日だけドットが灯る — 継続のモチベーションを静かに可視化。

Result

リリース後20万ダウンロード突破。App Store評価4.9は同カテゴリ最高水準。

最も嬉しかった指標は「毎日書くユーザーが62%」という数字。書く習慣をデザインの力で作れたことの証明だった。
